\section{Vect (Espaços Vetoriais Finitos sobre $\mathbb{C}$)}
\label{sec:vect}

Nesta categoria, os objetos são espaços vetoriais de dimensão finita sobre o corpo dos números complexos. Os morfismos são aplicações lineares entre esses espaços.

\subsection{Objetos}
Os objetos são representados pela sua dimensão e uma base. A base é tratada explicitamente como uma matriz cujas colunas são os vetores da base.

\subsection{Morfismos}
Um morfismo $f: V_1 \to V_2$ é uma aplicação linear representada por uma matriz $A$ de dimensão $|V_2| \times |V_1|$, tal que $f(v) = A \cdot v$ para todo $v \in V_1$.

\begin{lstlisting}[style=matlab, caption={Exemplo de Espaço Vetorial e Morfismo}]
	% Definição de um espaço vetorial V1 de dimensão 2
	V1.dim = 2;
	V1.basis = [1 0; 0 1];
	
	% Definição de um espaço vetorial V2 de dimensão 2
	V2.dim = 2;
	V2.basis = [1 0; 0 1];
	
	% Morfismo f: V1 -> V2 (matriz identidade)
	A = [1 0; 0 1];
\end{lstlisting}
