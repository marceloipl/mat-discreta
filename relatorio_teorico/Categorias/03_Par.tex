\section{Par (Aplicações Parciais)}

Semelhante a \textbf{Fin}, mas as funções podem não estar definidas para todos os elementos do domínio.

\subsection{Objetos}
Os objetos são definidos pela cardinalidade $n$ do conjunto.

\subsection{Morfismos}
Representados por vetores de comprimento $n$, utilizando o valor \texttt{0} (ou \texttt{NaN}) para elementos onde a função é indefinida.

\begin{lstlisting}[caption={Morfismo Parcial}]
% O valor 0 indica que o elemento não tem imagem
f = [1, 0, 2]; 
\end{lstlisting}
